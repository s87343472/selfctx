# SelfCtx · 个人上下文资产体系

> 让你在任意AI模型里都能被"认识"。
> 模型无关 · 本地存储 · 分级披露 · 15分钟上手

---

## 为什么需要这个

每次切换AI模型（Claude→GPT→Gemini→Grok），或者开启新对话，所有上下文归零。你要重新解释自己是谁、在做什么、有什么偏好——平均花30-50分钟，还找不齐信息。

SelfCtx的解法很简单：把"你是谁"存成一组本地Markdown文件，随时复制粘贴进任意模型。

---

## 快速开始

**第一步：导出草稿**

把 `prompts/P1_export.md` 里的prompt粘贴进你**已经积累了上下文**的模型（Claude/GPT等），让它帮你生成初版草稿。

**第二步：审查草稿**

检查输出里的 ⚠️ 标记，删除不想保留的内容。

**第三步：填入模板**

把审查后的内容填进 `templates/pca_template.md`，空白的地方标 `[待填写]`。

**第四步：选择导出级别**

从 `exports/` 文件夹选对应级别，复制进新模型：
- `L0_public.md` — 任意场景，无敏感信息
- `L1_work.md` — 日常工作协作（推荐起点）
- `L2_trusted.md` — 长期主力模型

**第五步：导入新模型**

把 `prompts/P2_import.md` 里的导入prompt + 你的PCA内容一起粘贴进新模型。

## 文件结构

```
selfctx/
├── README.md                 # 本文件
├── schema.md                 # 10个维度的完整字段定义
│
├── templates/
│   └── pca_template.md       # 空白填写模板（从这里开始）
│
├── prompts/
│   ├── P1_export.md          # 从当前模型导出草稿
│   ├── P2_import.md          # 导入新模型（全量）
│   ├── P3_import_quick.md    # 导入新模型（只加载行为规则）
│   ├── P4_update.md          # 更新单个维度
│   ├── P5_review.md          # 季度检视
│   └── P6_level_select.md    # 选择导出级别
│
├── exports/                  # 你的PCA导出文件（从模板生成后放这里）
│   ├── L0_public.md          # [你来填]
│   ├── L1_work.md            # [你来填]
│   ├── L2_trusted.md         # [你来填]
│   └── L3_full.md            # [你来填，永不外发]
│
└── my_pca/                   # 你的原始PCA文件（按维度分文件）
    ├── d1_narrative.md
    ├── d2_cognitive_os.md
    ├── d3_embodied.md
    ├── d4_emotional.md       # 可选
    ├── d5_capabilities.md
    ├── d6_assets.md
    ├── d7_relationships.md
    ├── d8_projects.md
    ├── d9_interests.md
    └── d10_worldview.md
```

## 10个维度

| # | 维度 | 核心问题 | 更新频率 | 必填 |
|---|------|---------|---------|------|
| D1 | 叙事身份 | 我是谁，怎么走到现在 | 年级 | ✓ |
| D2 | 认知操作系统 | 我怎么思考，对AI有什么要求 | 季度 | ✓ |
| D3 | 具身状态 | 我的身体和健康 | 月级 | ✓ |
| D4 | 情绪与心理 | 我的心理状态 | 月级 | 可选 |
| D5 | 能力与专长 | 我能做什么 | 季度 | ✓ |
| D6 | 资产图谱 | 我拥有什么 | 季度 | ✓ |
| D7 | 关系网络 | 我和谁有关系 | 季度 | ✓ |
| D8 | 项目快照 | 我在做什么（3-12个月窗口） | 月级 | ✓ |
| D9 | 兴趣与审美 | 我喜欢什么 | 年级 | ✓ |
| D10 | 世界观与志向 | 我相信什么，想去哪里 | 年级 | ✓ |

## 披露级别

| 级别 | Token预算 | 包含内容 | 适用场景 |
|------|---------|---------|---------|
| L0 公开版 | <500 | D1标签 + D2规则 + D9片段 | 任意模型，随意粘贴 |
| L1 工作版 | <2000 | D1+D2+D5+D7工作+D8+D9 | 日常AI工作协作 |
| L2 信任版 | <5000 | 全部维度，D3/D4/D6脱敏 | 长期主力模型 |
| L3 全量版 | 不限 | 全部维度，原始数据 | 仅本地，永不外发 |

---

## 维护节奏

不需要每天维护。建议：

- **每月**：检查D3（健康）和D8（项目快照）是否需要更新
- **每季度**：运行P5季度检视prompt，扫描所有维度
- **有重大变化时**：随时更新对应维度

## Claude用户

如果你主要使用Claude，可以安装 `selfctx.skill`，获得：
- 自动触发历史对话搜索
- 结构化导出，无需手动粘贴prompt
- 维度更新的内联支持

`.skill` 文件在 [releases] 页面下载。
